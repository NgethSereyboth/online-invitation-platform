# Development Phase — Canvas System, Reusable Groups, Background Images, Masks and Palettes

## Added
- Canvas zoom from 50% to 200%, fit-to-window, pan mode, and Space-drag panning.
- Optional visual rulers and safe-area guides.
- Zoom-aware dragging, resizing, marquee selection, snapping, alignment, and selection bounds.
- Reusable user-saved element groups stored in the browser and insertable into future designs.
- Section background images with cover/contain fitting and adjustable dark overlay.
- One-click use of a selected image object as a section background.
- Image masks: circle, arch, diamond, hexagon, and organic blob.
- Image frames: white, gold, and dark.
- Global invitation palette presets plus custom page, surface, text, heading, and accent colors.
- Published invitation and server public page support the new section backgrounds, masks, frames, and palette settings.
- Backend validation for all newly persisted design fields.

## Notes
Reusable element groups are browser-local in this phase. A later production phase can sync them to user accounts without changing their serialized object format.
